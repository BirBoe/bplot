\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T = Mono$>$\newline
class Axis$<$ T $>$}
The \mbox{\hyperlink{class_axis}{Axis}} class represents a (horizontal or vertical) axis that is part of a \mbox{\hyperlink{class_plot2_d}{Plot2D}}. The \mbox{\hyperlink{class_axis}{Axis}} consists of elements of the axis itself (e.\+g. \char`\"{}\+\_\+\char`\"{} or \char`\"{}$\vert$\char`\"{}) and may have labels (\char`\"{}x axis\char`\"{}) and ticks (\char`\"{}1 2 3 4\char`\"{}) to it. An \mbox{\hyperlink{class_axis}{Axis}} object cannot exists on its own but is always associated to a \mbox{\hyperlink{class_plot2_d}{Plot2D}}. 
\begin{DoxyTemplParams}{Template Parameters}
{\em T1} & Color model for this image (default\+: \mbox{\hyperlink{struct_mono}{Mono}}). \\
\hline
{\em T2} & Type of symbols that represent a \mbox{\hyperlink{class_pixel}{Pixel}} (Default\+: char). \\
\hline
\end{DoxyTemplParams}
