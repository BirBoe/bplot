\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T1 = Mono, typename T2 = char$>$\newline
class Image$<$ T1, T2 $>$}
The \mbox{\hyperlink{class_image}{Image}} class represents an image that can be printed to the console (or another output stream). The \mbox{\hyperlink{class_image}{Image}} is a collection of \mbox{\hyperlink{class_pixel}{Pixel}} objects that are ordered in a width x height \char`\"{}matrix\char`\"{}. The \mbox{\hyperlink{class_image}{Image}} can be either black and white (\mbox{\hyperlink{struct_mono}{Mono}}) or colored (\mbox{\hyperlink{struct_r_g_b}{RGB}}).
