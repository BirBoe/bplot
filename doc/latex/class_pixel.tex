\subsubsection*{template$<$typename T1 = Mono, typename T2 = char$>$\newline
class Pixel$<$ T1, T2 $>$}
The \mbox{\hyperlink{class_pixel}{Pixel}} class represents the smallest unit of an \mbox{\hyperlink{class_image}{Image}} and as such can be printed to the console (or another outpus stream). A \mbox{\hyperlink{class_pixel}{Pixel}} is represented as a symbol (for example a char, string or int) of a given color (\mbox{\hyperlink{struct_mono}{Mono}} of \mbox{\hyperlink{struct_r_g_b}{RGB}}).

\hline
