Detailed Description

template<typename T1 = Mono, typename T2 = char>
class Pixel< T1, T2 >

The Pixel class represents the smallest unit of an Image and as such can be printed to the console (or another outpus stream). A Pixel is represented as a symbol (for example a char, string or int) of a given color (Mono of RGB).

Template Parameters
T1Color model for this Pixel (default: Mono).
T2Type of symbols that represent the Pixel in the command line (Default: char).
