Detailed Description

template<typename T = Mono>
class Axis< T >

The Axis class represents a (horizontal or vertical) axis that is part of a Plot2D. The Axis consists of elements of the axis itself (e.g. "_" or "|") and may have labels ("x axis") and ticks ("1 2 3 4") to it. An Axis object cannot exists on its own but is always associated to a Plot2D.

Template Parameters
T1Color model for this image (default: Mono).
T2Type of symbols that represent a Pixel (Default: char).
